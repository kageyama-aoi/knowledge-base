3. 各サービス詳細

■ Vercel（Hobby）

できること

  • GitHub連携で自動デプロイ
  • フロントエンド公開
  • サーバーレス関数利用
  • Edgeリクエスト最大1,000,000
  • ビルド月6,000分

強み

  • 開発→公開が最速
  • プレビューURL自動生成
  • フロントに最適化

制約

  • 関数実行時間最大60秒
  • 重い処理は不向き
  • チーム機能制限あり

■ Cloudflare（Free）

Workers

  • 100,000リクエスト/日
  • 約10ms CPU時間
  • KVストレージ1GB

Pages

  • 月500ビルド
  • 静的サイト向き
  • CDN高速配信

強み

  • 世界規模の高速配信
  • 軽量APIに最適
  • セキュリティ強い

制約

  • CPU時間が短い
  • 重いDB処理は難しい

■ Supabase（Free）

できること

  • PostgreSQL DB（500MB）
  • 認証機能
  • API自動生成
  • ストレージ1GB
  • MAU 50,000

強み

  • 本格RDB
  • Row Level Security
  • バックエンド一式揃う

制約

  • 2プロジェクトまで
  • 7日低活動で停止
  • DB設計力が必要
