# Vercel / Cloudflare / Supabase  
## 無料プラン比較＆詳細整理（基準日：2026-02-21）

---

# 1. まずは直感比較表

| 項目 | Vercel (Hobby) | Cloudflare (Free) | Supabase (Free) |
|------|----------------|-------------------|-----------------|
| 主な役割 | フロント公開 | CDN＋エッジ実行 | バックエンド（DB＋認証） |
| 何を解決する？ | 画面を即公開 | 世界中に速く届ける | DB・ログインを即用意 |
| 得意分野 | フロント特化 | 配信・軽量処理 | データ管理 |
| コード実行場所 | サーバーレス（Vercel側） | 世界中のエッジ | サーバー側（DB中心） |
| 無料での強み | Git push → 即公開 | 100kリクエスト/日 | Postgres本物DB |
| 無料での壁 | 関数実行時間制限 | CPU 10ms制限 | 500MB制限／停止あり |
| 向いている人 | UIを早く試したい | パフォーマンス重視 | DB設計できる人 |

# 3. 各サービス詳細

---

## ■ Vercel（Hobby）

### できること
- GitHub連携で自動デプロイ
- フロントエンド公開
- サーバーレス関数利用
- Edgeリクエスト最大1,000,000
- ビルド月6,000分

### 強み
- 開発→公開が最速
- プレビューURL自動生成
- フロントに最適化

### 制約
- 関数実行時間最大60秒
- 重い処理は不向き
- チーム機能制限あり

---

## ■ Cloudflare（Free）

### Workers
- 100,000リクエスト/日
- 約10ms CPU時間
- KVストレージ1GB

### Pages
- 月500ビルド
- 静的サイト向き
- CDN高速配信

### 強み
- 世界規模の高速配信
- 軽量APIに最適
- セキュリティ強い

### 制約
- CPU時間が短い
- 重いDB処理は難しい

---

## ■ Supabase（Free）

### できること
- PostgreSQL DB（500MB）
- 認証機能
- API自動生成
- ストレージ1GB
- MAU 50,000

### 強み
- 本格RDB
- Row Level Security
- バックエンド一式揃う

### 制約
- 2プロジェクトまで
- 7日低活動で停止
- DB設計力が必要
